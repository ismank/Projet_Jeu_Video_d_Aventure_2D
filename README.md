# 🎮 **Jeu d'Aventure 2D en C++**

## 📌 Description
Jeu vidéo d’aventure 2D développé en **C++** dans le cadre du **BUT Informatique**.  
Le projet consiste à programmer différentes mécaniques de gameplay (logique du jeu, interactions, déplacements) et a été réalisé en binôme avec des phases de **tests, correction de bugs et gestion de projet**.

## 📚 Objectifs du projet
- Développer un **jeu vidéo** en 2D
- Implémenter différentes fonctionnalités de gameplay
- Travailler en collaboration (binôme)
- Mettre en place des phases de tests et correction d’erreurs
- Appliquer des compétences en programmation et gestion de projet

## ⚙️ Technologies utilisées
- **C++**

## 🎮 Fonctionnalités
- Déplacement du personnage
- Interactions avec l’environnement
- Gestion des évènements
- Gestion de la logique du jeu
- Gameplay simple
- Tests et correction des erreurs

## 👥 Équipe
Projet réalisé en binôme dans le cadre d’un projet universitaire.
